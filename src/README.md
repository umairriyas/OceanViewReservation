# OceanView Hotel Reservation System

A web-based hotel reservation management system built with:
- Java Servlets & JSP
- MySQL Database
- Apache Tomcat 10.1
- MVC Architecture

## Features
- Staff Login with Session Management
- Add New Reservations
- View All Reservations
- Calculate & Print Guest Bill
- Delete Reservations

## Setup
1. Import project in IntelliJ IDEA
2. Run MySQL script to create database
3. Deploy on Apache Tomcat 10.1
4. Access at http://localhost:8080/OceanViewReservation
